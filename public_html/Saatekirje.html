Tervetuloa käyttämään Score-app -sovellustamme.

Sovellus oli alunperin OAMK:n ja FONTYS:n yhteistyöprojekti. OAMK:n opiskelijoiden tarkoituksena oli tehdä front-end, ja FONTYS:n opiskelijoiden back-end. Aiheeksi valittiin pisteidenlaskusovellus, joka ei ota kantaa lajiin, jonka pisteitä lasketaan. Kurssi päättyi keväällä 2016.

OAMK:n opiskelijat päättivät jatkaa sovelluksen tekoa kurssin päätyttyä. Kirjoitimme uuden back-endin puhtaalta pöydältä. Sovellus on tällä hetkellä alkuperäisen kurssin sprint1:n tehtävälistan mukainen. Edessäsi on alfaversio 0,1.

Tällä hetkellä toteutettuja ominaisuuksia:

  • Uuden käyttäjän rekisteröityminen
  • Korkeintaan 2 ihmisen game roomien luonti
  • Vapaisiin game roomeihin liittyminen
  • Itse luotujen game roomien pisteiden hallinta
  • Privaattipelejä, jotka eivät näy muille kuin hallitsijalle ja osallistujille
  • Osittainen käännös. OAMK:n kurssilla toteutettu. Uudet osat ovat kääntämättä.
  • Osittainen mobiilituki. Toimii mobiililla, mutta UI ei ole suunniteltu mobiilille.

Suunnitteilla:

  • Otetaan käyttöön HTTPS
  • Otetaan käyttöön jwt-web tokenit
  • Salataan ihmisten salasanat tietokannassa
  • Käyttäjäryhmien luonti
  • Lajityypin lisääminen game roomeihin
  • Muutetaan käyttäjien liittäminen kutsumiseksi. Kutsuttu hyväksyy
  • Useampien pelaajien game roomit
  • Game roomeista muodostuvat turnaukset
  • Turnauksista muodotuvat liigat
  • Salasanan vaihto
  • Profiilien kustomointi
  • Hallinnoitujen pelien täysi kustomointi. Tällä hetkellä pystyy vaihtamaan vain pisteitä
  • Game roomien poistaminen
  • Lomake palautteen antamiseen sivuille

Jos olet kiinnostunut osallistumaan sovelluksen kehittämiseen ole yhteydessä. Mahdolliset bugit tai serverin kaatumisen voi ilmoittaa Villelle tai Annille.

Olisimme kiitollisia palautteesta.
