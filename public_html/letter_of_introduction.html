Welcome to Score-app.

The app started as a collaboration project between OAMK and FONTYS. The OAMK students were responsible for the frontend and the FONTYS students were responsible for the backend. We chose to do an score keeping application. The application was supposed to be applicable for all kinds of games, not just for sports. The course ended in the spring of 2016.

The OAMK students decided to continue building the application. We wrote a completely new backend. We took as our goal the original projects sprint one task list. You have in front of you our alpha version 0,1.

Currently implemented:

  • Creating a user and logging in
  • Game rooms of up to two people
  • Joining non private game rooms
  • Limited managing of created game rooms. Changing the score.
  • Partial translation. Parts from the original project are done and the new stuff is only in English.
  • Partial mobile support. Ui is not designed for mobile.

Currently planned:

  • Use HTTPS instead of HTTP
  • Use jwt web tokens
  • Encrypt user passwords in the database
  • Create user groups
  • Adding type to game rooms
  • Change adding users to inviting users in game rooms
  • Game rooms with more than 2 players
  • Tournaments formed of game rooms
  • Leagues formed of tournaments
  • Changing of passwords
  • Customising of user profiles
  • Full managing of created game rooms
  • Deleting game rooms
  • A form for giving feedback to the developers

If you are interested in taking part of the development, get in tough. You can report possible bugs and server problems to Ville or Anni.

We would be crateful of any feedback you can give us.
